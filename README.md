Brotato Mod that adds Kirby as a new playable character

### Features
* Kirby as a playable character  
* Inhale weapon
  * Size grows with range stat
  * Knockback increases with range stat
  * Knockback is reversed (pulls enemies to you)
  * Pulls items to you
